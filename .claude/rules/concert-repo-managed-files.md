## Concert Repo: ALL Files Are Freely Editable

This is the Concert source repository. **You are allowed to edit every file in this repo**, including files with the header `AUTO-GENERATED BY CONCERT — DO NOT EDIT`.

The "do not edit" warning applies only to **target repos** where these files are deployed by `concert init` or `concert update`. In this repo — the concert repo itself — these files are the source of truth and are meant to be edited directly.

Do NOT skip or avoid editing any file in this repo due to "do not edit" headers or "managed" path concerns. All of the following paths (and any other path) are fully editable here:

- `.claude/agents/`
- `.claude/skills/`
- `.claude/commands/concert/`
- `.claude/rules/`
- `.github/agents/`
- `.github/workflows/concert-*.yml`
- `.concert/workflows/`
- `.concert/stage-registry.jsonc`
- `.concert/templates/`
- `concert.jsonc`
