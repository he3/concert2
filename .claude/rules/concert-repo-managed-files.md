## Concert Repo: Managed Files Are Editable

This is the Concert source repository. Files with the header `AUTO-GENERATED BY CONCERT — DO NOT EDIT` are the **source of truth** here — they are meant to be edited directly.

The "do not edit" warning applies only to **target repos** where these files are installed by `concert init` or `concert update`. In this repo, edit them freely.

This applies to all files in:
- `.claude/agents/`
- `.claude/skills/`
- `.claude/commands/concert/`
- `.github/agents/`
- `.github/workflows/concert-*.yml`
- `docs/concert/workflows/`
