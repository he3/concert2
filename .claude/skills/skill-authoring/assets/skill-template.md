## Standard SKILL.md Runbook Sections

When generating a new SKILL.md, use these sections to ensure reliability:

- **What it does:** Describe when the skill should be used.
- **Inputs needed:** State required paths, URLs, IDs, and credentials.
- **Workflow:** Numbered steps and exact commands.
- **Output format:** Strict guidelines so results look the same every run.
- **Guardrails:** Rules for safety and hallucination control.
- **Failure handling:** What to do when a command fails or data is missing.
