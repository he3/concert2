## Objective

Write a validation test suite that ensures all template files in the `templates/` directory are present, well-formed, and internally consistent. This test catches missing files, broken references, and format errors before the package is published.

## Files

- `src/__tests__/templates.test.ts`

## Requirements

- FR-001: Repository Bootstrapping via npx
- FR-012: Managed File Headers
- FR-025: npm Package Distribution

## Detailed Instructions

### src/__tests__/templates.test.ts

Write a test file that validates the `templates/` directory. Use `node:fs` and `node:path` to read and check files. The test file should use `describe`/`it` structure.

**Test groups:**

#### 1. Directory structure exists

```typescript
// Verify all expected directories exist
const expectedDirs = [
  "templates/docs/concert/agents",
  "templates/docs/concert/workflows",

  "templates/.github/workflows",

// For each: assert fs.existsSync(path.resolve(ROOT, dir))
```
